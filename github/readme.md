# Sam Auerbach's Portfolio
This is the build file/repository for Sam Auerbach's personal portfolio hosted by [GitHub Pages](https://pages.github.com/) at [samrauerbach.github.io](https://samrauerbach.github.io).

## Source files
The source files for this website can be found in my [portfolio repository](https://github.com/samrauerbach/portfolio).

> This is a static website built using [Vue.js](https://v3.vuejs.org/) and [Quasar](https://quasar.dev/).
>
> This repository was deployed using a bash script that takes the build output and pushes it to this repository, which triggers GitHub-Pages to redeploy the website.
